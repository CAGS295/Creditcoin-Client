# Introduction 
Command-line client for the Creditcoin blockchain. 

# Getting Started
The Creditcoin Client docker images are available on [Docker Hub](https://hub.docker.com/r/gluwa/creditcoin-client).

# Build and Test
```
dotnet restore
dotnet build
```
